Einführung

Hier eine kurze und knackige Einführung in das Spiel Mastermind:

1. Der Codemacher erstellt zu erst seinen geheimen Code, zum Beispiel: grün blau gelb rot

2. Nachdem der Code erstellt wurde, ist der Codebrecher an der Reihe. Er muss den Code innerhalb 7 Runden herausfinden.

3. Nach jeder Runde bekommt der Codebrecher Rückmeldung ob der eingegebene Code korrekt war. Der Codemacher hat die Möglichkeit seine Rückmeldung mithilfe weißer und pinker (da man schwarz nicht gut sieht) Punkten darzustellen:

weiß = farblich richtig jedoch an der falschen Position

pink = Farbe und Position sind richtig
