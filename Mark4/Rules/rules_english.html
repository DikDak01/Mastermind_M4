Introduction

Here is a short and sweet introduction to the game Mastermind:

1. The code maker first creates his secret code, for example: green blue yellow red

2. After the code is created, it is the turn of the code breaker. He has to find out the code within 7 rounds.

3. After each round, the code breaker receives feedback as to whether the code entered was correct. The code maker has the option of displaying his feedback using white and pink (since you can't see black well) dots:

white = correct color but in the wrong position

pink = Color and position are correct
